Enigma Kiosk

A web app created for HackChicago 2018. Our web app utilizes speech detection technology and natural language processing algorithms to recognize a customer's order at a restaurant from diction. The web app then saves the data and displays it for the user to see and check out. The application was written to be scalable and most components are not hard coded but rather interpolated from the server. The application was written mainly in Python and Javascript

PiDog

An android phone application that won 3rd place at Chicagohacks 2017. The application allows a user to operate a self driving robotic car using voice recognition. Anyone with a sight or navigation disability can use this application to say something as “Find bench” and the car -- that was build from scratch -- will use feed forward neural networks to navigate towards the image that the program has been trained to recognize. The application was written in Java and Python.
